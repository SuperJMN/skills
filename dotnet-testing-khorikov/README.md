# dotnet-testing-khorikov

Codex/GPT-5.5 optimized skill for writing and reviewing .NET tests using Khorikov-style principles.

## Layout

- `SKILL.md`: short operational skill loaded by Codex when relevant.
- `references/`: optional deeper material for edge cases and reviews.
- `scripts/`: optional helper scripts for deterministic verification.

## Install

Copy the `dotnet-testing-khorikov` directory into your Codex skills directory or your repository's skills folder, depending on your setup.

The important entrypoint is `SKILL.md`.
